# **App Name**: Event Reporter

## Core Features:

- Event Category Selection: Allow the user to select a category of events from a predefined list, as depicted in the image.
- Event Reporting: Generate a report based on the selected event category. Reports are custom made to correspond to each category of event.
- Search Functionality: Implement a search bar to allow users to quickly find specific event categories.

## Style Guidelines:

- Primary color: Light blue (#E0F2F7), providing a clean and calming backdrop for event reporting.
- Background color: Almost white (#F9FBFC), offering subtle contrast with the tiles and avoiding pure white.
- Accent color: Purple (#A090E0), used for highlighting interactive elements such as the search bar or selected event types.
- Body and headline font: 'PT Sans', a humanist sans-serif combining modernity with a touch of warmth, ideal for clarity and readability.
- Use distinct and symbolic icons for each event category to ensure easy recognition and navigation.
- Tile-based grid layout for presenting event categories, mimicking the clean and structured design of the original image.
- Subtle animations or transitions upon selecting an event category to enhance user engagement and provide feedback.
